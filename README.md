# Python Project

## Requirements

Python 3.10+

## Installation

Install the required packages in your local environment

```bash
pip install -r requirements
```

## Setup

1. mkdir ./out
2. mkdir ./in

## Run It

```bash
python main.py {file: ex(in/test.txt OR http://____.com/test.txt)}
```

## Run Tests

```bash
pytest
```
